This folder contains zip files pertaining to the re-run of the nr. servers-rep. factor experiment of Milestone 2 in Section 2 of Milestone 3.

memaslap_logs.zip contains the memaslap logs of the clients. Each log is of the form log_{client}_{nr. servers}_{rep. factor}_{repetition}_105s.

mw_logs_{nr. servers}.zip contains the middleware logs of the experiments with {nr. servers} servers. Each log is of the form log_{nr. servers}_{rep. factor}_{repetition}.log.

proc_memaslap.zip contains processed memaslap data (TPS, TPS STDev, RT, RT STDev) for GET and SET requests, and also overall.
